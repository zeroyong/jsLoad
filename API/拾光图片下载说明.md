## 概述

拾光壁纸(gallery.timeline.ink) API返回的图片URL是经过AES加密的，需要使用对应的rawprovider作为密钥进行解密后才能正常访问。

## 2. URL解密

### 核心逻辑

API返回的imgurl和thumburl是加密的，需要使用`rawprovider`字段作为密钥进行AES-CBC解密。

### 解密流程

1. 提取文件名中的name_part（去掉扩展名和路径）
2. 如果name_part长度>=32，取前32字符为加密部分p0，后16字符为原始部分p1
3. 对p0进行AES解密
4. 拼接解密结果 + p1 + 扩展名

### Python实现

```python
import hashlib
import base64
from Crypto.Cipher import AES

def encrypt_md5(t):
    """MD5加密"""
    if not t:
        return ""
    return hashlib.md5(t.encode()).hexdigest()

def decrypt_aes(t, e):
    """AES解密 - CBC模式, ZeroPadding"""
    if not t or not e:
        return ""
    # 密钥: e重复16次取最后16位
    e_key = (e * 16)[-16:]
    # IV: MD5(e_key)[8:24]
    key_md5 = encrypt_md5(e_key)[8:24]
    
    hex_bytes = bytes.fromhex(t)
    b64_str = base64.b64encode(hex_bytes).decode()
    cipher = AES.new(e_key.encode(), AES.MODE_CBC, key_md5.encode())
    decrypted = cipher.decrypt(base64.b64decode(b64_str))
    result = decrypted.rstrip(b'\x00').decode('utf-8', errors='replace')
    return result

def decrypt_url(t, e):
    """解密完整的图片URL"""
    if not t or not e:
        return ""
    
    parts = t.split("?")
    path_parts = parts[0].split("/")
    filename = path_parts[-1].split(".")
    name_part = filename[0]
    
    if len(name_part) < 32:
        return t
    
    # 分割加密部分和原始部分
    p0 = name_part[:32]
    p1 = name_part[32:]
    
    # 解密
    p0_decrypted = decrypt_aes(p0, e)
    filename[0] = p0_decrypted + p1
    path_parts[-1] = ".".join(filename)
    parts[0] = "/".join(path_parts)
    return "?".join(parts)
```

### Flutter实现要点

Flutter需要使用以下包：
- `crypto` - MD5计算
- `encrypt` - AES解密
- `pointycastle` - 底层加密

```dart
// Dart实现伪代码
import 'package:crypto/crypto.dart';
import 'package:encrypt/encrypt.dart' as encrypt;

String encryptMd5(String t) {
  return md5.convert(utf8.encode(t)).toString();
}

String decryptAes(String t, String e) {
  String eKey = (e * 16).substring((e * 16).length - 16);
  String iv = encryptMd5(eKey).substring(8, 24);
  
  // Hex转Bytes
  List<int> hexBytes = hex.decode(t);
  // Base64编码
  String b64 = base64.encode(hexBytes);
  
  // 解密
  final cipher = encrypt.AES(
    encrypt.Key.fromUtf8(eKey),
    mode: encrypt.AESMode.cbc,
    iv: encrypt.IV.fromUtf8(iv),
  );
  final decrypted = cipher.decrypt(encrypt.Encrypted.fromBase64(b64));
  
  // 去除ZeroPadding (末尾的0)
  return decrypted.trimRight();
}
```

## 3. 图片下载

### 不同域名需要不同的Headers

```python
# ananas.chaoxing.com (超星云盘)
ananas_headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
    'Cookie': 'special_fontSize=16px; special_viewModel=dyb; special_local_version=1',
    'Sec-Fetch-Dest': 'document',
    'Sec-Fetch-Mode': 'navigate',
    'Sec-Fetch-Site': 'none',
    'Sec-Fetch-User': '?1',
    'Upgrade-Insecure-Requests': '1',
}

# i0.hdslb.com (B站图床)
hdslb_headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
    'Sec-Fetch-Dest': 'image',
    'Sec-Fetch-Mode': 'no-cors',
    'Sec-Fetch-Site': 'cross-site',
    'Accept': 'image/avif,image/webp,image/apng,image/svg+xml,image/*,*/*;q=0.8',
}

# c-ssl.duitang.com (堆糖)
duitang_headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
    'Referer': 'https://gallery.timeline.ink/',
}

def get_headers(img_url):
    if 'ananas.chaoxing.com' in img_url:
        return ananas_headers
    elif 'hdslb.com' in img_url:
        return hdslb_headers
    else:
        return duitang_headers
```

### 多线程下载

```python
from concurrent.futures import ThreadPoolExecutor, as_completed

MAX_WORKERS = 8

def download_img(item, save_dir):
    img_url = item.get('imgurl', '')
    img_id = item.get('id', 'unknown')
    
    resp = requests.get(img_url, headers=get_headers(img_url), timeout=30)
    ext = os.path.splitext(img_url)[1] or '.jpg'
    filename = f"{img_id}{ext}"
    
    with open(os.path.join(save_dir, filename), 'wb') as f:
        f.write(resp.content)
    return True

# 使用ThreadPoolExecutor并发下载
with ThreadPoolExecutor(max_workers=MAX_WORKERS) as executor:
    futures = [executor.submit(download_img, item, SAVE_DIR) for item in items]
    for future in as_completed(futures):
        future.result()
```

## 4. 完整流程

```python
def main():
    # 1. 获取图片列表
    items = get_image_list()  # 自动解密URL
    
    # 2. 多线程下载
    with ThreadPoolExecutor(max_workers=8) as executor:
```

## 5. 注意事项

1. **Timeline-Device**: 这是设备标识符，可以自行生成一个UUID，需要保持一致
2. **rawprovider**: 每个图片的rawprovider不同，解密时需要使用对应图片的rawprovider
3. **图片域名**: 图片可能来自不同域名(ananas/hdslb/duitang)，需要使用不同的headers
4. **错误处理**: 解密失败时返回原始URL（服务器可能不做严格验证）
5. **数量限制**: no参数控制返回数量，最大约68张
