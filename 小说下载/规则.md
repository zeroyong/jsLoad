用你已有的信息，封面规则可以这样写：

**直接填在阅读App的封面规则框里**（不用转义）：

```
.hot_sale a@href##/shoujixs_(\d{3})(\d+)/##https://images.shoujixsw.com/$1/$1$2/$1$2s.jpg
```

**如果写到书源JSON中**（需要转义反斜杠）：

```json
"coverUrl": ".hot_sale a@href##/shoujixs_(\\d{3})(\\d+)/##https://images.shoujixsw.com/$1/$1$2/$1$2s.jpg"
```

---

### 解析一下这条规则做了什么：

1. `.hot_sale a@href`  
   先获取到书籍ID：`/shoujixs_146258/`

2. `##/shoujixs_(\d{3})(\d+)/`  
   用正则把它拆成两组：  
   - 第一组 `(\d{3})` 匹配前三位 `146`  
   - 第二组 `(\d+)` 匹配剩余数字 `258`  
   整个 href 被这段正则完全匹配。

3. `##https://images.shoujixsw.com/$1/$1$2/$1$2s.jpg`  
   用捕获组拼接出封面地址：  
   - `$1` = `146`  
   - `$1$2` = `146258`  
   结果：`https://images.shoujixsw.com/146/146258/146258s.jpg`

这样就一步到位从书籍ID生成了封面URL，无需中间规则。
